This is a realtime bus tracker for Philly's SEPTA busses. 

By default it only displays the 21 and 42 (which are the busses I mostly ride). 

However, you can manually add other routes from within the map as needed. 

You can also filter by bus direction, for example, if you only want to see the west-bound busses, etc.

It is currently running on Netlify.
